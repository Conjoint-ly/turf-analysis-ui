`Функциональный компонент`

# Поток данных

---

## Параметры

**`currentPage`**  
Тип: число

Текущая страница.

**`pageSize`**  
Тип: число

Количество страниц.

**`showingItems`**  
Тип: число

Количество отображаемых строк данных.

**`totalItems`**
Тип: число

Общее количество строк данных.
